# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Navigation with smooth scroll behavior
- **Interactive Elements**: Hover effects, animations, and transitions
- **Contact Form**: Functional contact form (ready for backend integration)
- **Mobile Menu**: Hamburger menu for mobile navigation

## Sections

1. **Hero Section**: Introduction with call-to-action buttons
2. **About**: Personal information and statistics
3. **Skills**: Technical skills organized by category
4. **Projects**: Showcase of featured projects
5. **Contact**: Contact form and social media links

## Getting Started

1. Open `index.html` in your web browser
2. No build process or dependencies required - it's pure HTML, CSS, and JavaScript!

## Customization

### Update Personal Information

1. **Name**: Replace "Ayan Taghiyeva" throughout the HTML file
2. **Contact Details**: Update email, phone, and location in the contact section
3. **Social Links**: Add your actual GitHub, LinkedIn, and Twitter URLs
4. **Projects**: Replace placeholder projects with your actual work
5. **Skills**: Update the skills list to match your expertise
6. **About Section**: Customize the about text and statistics

### Styling

- Colors can be customized in the `:root` CSS variables in `styles.css`
- Fonts can be changed by updating the Google Fonts import in `index.html`
- Layout adjustments can be made in the respective section styles

### Adding Your Photo

Replace the SVG placeholder in the hero section with your actual photo:

```html
<img src="your-photo.jpg" alt="Your Name" class="hero-photo">
```
